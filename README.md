## 🇺🇸 About this Repository (English)

Repository dedicated to the exercises and the final project developed during the **"Python: Create your first application"** course, provided by Alura. 

This module focuses on building a solid foundation in Python programming, covering everything from basic syntax to essential data structures and control flow. 

### 🧠 Concepts & Skills Applied
Based on the scripts developed in this repository, the following core concepts were mastered:
- **Fundamentals & I/O:** Basic syntax, variables, and terminal interactions (`print.py`, `app.py`).
- **Control Flow:** Conditional logic (`idade.py`, `parimpar.py`).
- **Loops:** Iteration and repetition structures (`for.py`, `while.py`).
- **Data Structures:** Data manipulation using Lists (`listas.py`, `listasex...`) and Dictionaries (`diciex...`).
- **Mathematical Logic:** Coordinate systems and numerical operations (`cordenada.py`).
